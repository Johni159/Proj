# Proj
Консольний трекер витрат для контролю бюджету
Розробив застосунок для автоматизації особистих фінансів (Python). Реалізував структуру зберігання даних (JSON), систему фільтрації за категоріями та генерацію звітів. Використовував Cursor/AI для прискорення розробки та дотримання best practices написання коду.
Технології: Python, JSON.
Як запустити: Коротка інструкція (команди в терміналі).
